### 1. Перечень автоматизируемых сценариев

#### *Набор сценариев для автоматизированного тестирования.*

#### Валидные данные для заполнения полей формы (актуальны для всех сценариев)
* Поле "Номер карты" заполняется цифрами в количестве 16-ти символов (при генерации случайного значения устаналивается плагин Faker, с помощью которого описывается класс, возвращающий 16-ти значный параметр )
* Поле "Месяц" заполняется цифрами в количестве 2-х символов в диапазоне от 01 до 12, генерация значения будет отталкиваться от текущего месяца;
* Поле "Год" заполняется цифрами в количестве 2-х символов в диапазоне от 23 до 28, генерация значения будет отталикавться от текущего года;
* Поле "Владелец" заполняется латинскими буквами в количестве 64-х символов. Допускается использование пробела и дефиса.
* Поле "CVC/CVV" заполяется цифрами в количестве 3-х символов.

#### Предусловие (актуально для всех сценариев)
* Открыть браузер Chrome
* Перейдите по ссылке http://localhost:8080/
* Нажать кнопку "Купить" 

**1. Оплата по одобренной карте валидными данными с получением уведомления об успешной операции**
1. Ввести в поле "Номер карты" 4444 4444 4444 4441
2. Заполнить поле "Месяц" валидным значением
3. Заполнить поле "Год" валидным значением
4. Заполнить поле "Владелец" валидным значением
5. Заполнить поле "CVC/CVV" валидным значением
6. Нажать на кнопку Продолжить

*Ожидаемый результат:* 
Появление сообщения "Успешно! Операция одобрена Банком." Выполняется запись в БД статус "APPROVED"

### 2. Перечень используемых инструментов с обоснованием выбора
* Java 11 - язык программирования для написания кода приложения и автоматизированных тестов.
* Gradle - система автоматической сборки, используемая для подключения зависимостей, которые выкачивают нужные библиотеки.
* Git и GitHub - система контроля версий.
* Faker - библиотека для генерирования фейковых данных, используемых для прогона тестов.
* Lombok - библиотека Java, позволяющая сократить шаблонный код.
* Selenide - фреймворк для автоматизированного тестирования веб-приложений.
* Docker - система контейнеров для запуска эмуляторов MySQL и PostgreSQL.
* Allure Report - система подготовки отчетов.
